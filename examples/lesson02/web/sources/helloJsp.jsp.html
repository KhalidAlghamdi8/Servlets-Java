<%@page contentType="text/html" pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title>JSP helloJsp</title>
  </head>
  <body>
    <h1>HELLO JSP!</h1>
    <h2>${1+1}</h2>
  </body>
</html>
A Java Server Pages (JSP) file is usually an HTML file with special markup or elements that allows you to print dynamic data using Java or Expression Language syntax.
In this example, Expression Language is used to print the result of the operation 1+1 using: ${1+1} JSP allows developers to print web pages to the user in a better and more maintainable way than servlets.
Note that JSP can be used with servlets to balance the Java code in servlets and display logic in the JSP pages.
