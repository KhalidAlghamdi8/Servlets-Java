<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title>Page 2</title>
    <link rel="stylesheet" href="res/styles.css" type="text/css"> 
  </head>
  <body>
    <jsp:include page="page_header.jsp"/>
    <h1>Welcome to page 2!</h1>
    <jsp:include page="page_footer.jsp"/>
  </body>
</html>
In this example, filters are used to add data to the request to provide the information required for the header and footer in the page.
This is particularly useful since this data is presented in every page that uses the template files. Instead of using parameters or having a servlet, this information is placed on the request for using filters to include the data that is displayed in the pages.
Note that in this page the header and footer are included without parameters and request attributes are set in the page.
