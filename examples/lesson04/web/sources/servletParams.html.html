In this form, 2 parameters are sent to the servlet. If the checkbox is not checked, that parameter is not sent.
In this form there are some checkboxes. Note that the value of the checkboxes is sent as the parameter value if the checkbox is checked, otherwise the entire parameter is not sent.
When multiple options are valid for one parameter, use the same parameter name for each input entry: 
<input type="checkbox" name="color" value="blue"/>
<input type="checkbox" name="color" value="green"/>
<input type="checkbox" name="color" value="red"/>
In this form the behavior of text fields is tested. Note that, like the checkboxes, if a text field is left empty, the parameter is not sent to the servlet.
In this form there are some radio buttons. The value sent in the color parameter is the one of the selected radio button. If none is selected, no value is sent to the servlet.
There 3 hidden fields, note that all 3 get to the servlet although the third one has no value.
