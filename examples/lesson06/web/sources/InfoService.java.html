A good way to do dependency injection is to creating interfaces for the business objects.
Abstract only the methods that are required, In the following example, the interface specifies only the required methods.
Injecting this interface using the @Inject annotation injects the correct implementation. The server finds the implementation by looking at the annotations in implementing classes.
