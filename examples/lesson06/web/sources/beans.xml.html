<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://java.sun.com/xml/ns/javaee"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://java.sun.com/xml/ns/javaee http://java.sun.com/xml/ns/javaee/beans_1_0.xsd">
</beans>
The /WEB-INF/beans.xml file tells the server that it should use Context and Dependency Injection.
This file may contain the configuration for beans that can be created in the server.
In addition to the beans.xml configuration, classes annotated are also processed.
NOTE: Even if all objects are configured with annotations, this file is REQUIRED if you want injection to work. A simple "empty" file like this will work.
