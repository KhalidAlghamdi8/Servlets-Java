package com.examples.lesson05.listeners;

import javax.servlet.ServletRequestEvent;
import javax.servlet.ServletRequestListener;
import javax.servlet.annotation.WebListener;

@WebListener()
public class ExampleRequestListener implements ServletRequestListener {

  int requestCount = 0;

  @Override
  public void requestDestroyed(ServletRequestEvent sre) {
  }

  @Override
  public void requestInitialized(ServletRequestEvent sre) {
    requestCount++;
    sre.getServletRequest().setAttribute("requestCount", requestCount);
  }
}
This method is called when a request is created. It is a good place to set request attributes that are used by servlets or JSP views and might change over time.
In this case, this listener keeps track of the number of requests created.
This method is called when the request finishes and is destroyed is usually used for logging or cleanup purposes.
The is a web application life-cycle listener.
This listener is configured by adding the @WebListener annotation. Alternatively, you may add the listener element to the web.xml file: 
  <listener>
    <listener-class>LISTENER CLASS FULLY QUALIFIED NAME</listener-class>
  </listener>
