package com.examples.lesson05.listeners;

import javax.servlet.annotation.WebListener;
import javax.servlet.http.HttpSession;
import javax.servlet.http.HttpSessionAttributeListener;
import javax.servlet.http.HttpSessionBindingEvent;

@WebListener()
public class ExampleSessionAttributeListener implements HttpSessionAttributeListener {

  @Override
  public void attributeAdded(HttpSessionBindingEvent event) {
    HttpSession session = event.getSession();
    Integer count = (Integer) session.getAttribute("attributeCount");
    if (count == null) {
      count = 0;
    }
    count++;
    session.setAttribute("attributeCount", count);
  }

  @Override
  public void attributeRemoved(HttpSessionBindingEvent event) {
  }

  @Override
  public void attributeReplaced(HttpSessionBindingEvent event) {
  }
}
This method is called when an attribute is added to the session scope.
The goal of this method is to keep track of how many objects are in this scope.
The count is stored inside another attribute inside of the same scope.
This method is called when an attribute is removed from the session scope.
This method is called when an attribute is replaced by another. This occurs when a setAttribute() is called in an already set attribute.
The HttpSessionAttributeListener is a web application life-cycle listener.
This listener is configured by adding the @WebListener annotation. Alternatively, you can add the listener element to the web.xml file: 
  <listener>
    <listener-class>LISTENER CLASS FULLY QUALIFIED NAME</listener-class>
  </listener>
