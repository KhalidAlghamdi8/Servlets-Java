JSP XML documents are similar to scriptless JSP except that no scripts or directives are used. Instead, only JSP and some other tags are allowed. JSP documents are not uncommon, but it is still important to acknowledge their existence and how they work.
JSPX documents may be used for defining XML or HTML documents (HTML is a special form of XML).
The useBean tag creates or receives a bean from a specific scope. In this case, the nameBean request bean is created using the NameBean class.
You may set specific attributes to the bean. In the jsp:setAttribute tag, the name attribute is set using the request parameter 'name'.
To display the contents of a bean or to execute an expression that returns a value, the expression must be placed inside a jsp:expression tag, in a similar way to <%= JSP scriptlets.
