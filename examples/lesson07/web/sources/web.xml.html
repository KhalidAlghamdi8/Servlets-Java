To set up initialization parameters for JSP you have to use the web.xml file.
Note that you use: 
<jsp-file>/lifecycle.jsp</jsp-file>
to specify that the servlet refers to a JSP file.
Then specify the initialization parameters as with any other servlet. In this case the parameter "jsp-name" is set to "JSP Lifecycle" 
<init-param>  
  <param-name>jsp-name</param-name>  
  <param-value>JSP Lifecycle!</param-value>  
</init-param>
