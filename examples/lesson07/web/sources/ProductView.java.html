package com.examples.lesson07.view;

import com.examples.lesson07.model.Product;

public class ProductView {

  private final Product product;
  private final int quantity;

  public ProductView(Product product, int quantity) {
    this.product = product;
    this.quantity = quantity;
  }

  public Product getProduct() {
    return product;
  }

  public int getQuantity() {
    return quantity;
  }

  public double getTotalPrice() {
    return product.getPrice() * quantity;
  }
}
The ProductView class contains a product object and some additional information required by the JSP page to display.
In the constructor method you might notice that the original product is stored. The ordered quantity for that product is obtained and stored in this object.
The method getTotalPrice calculates the total price based on the quantity and the product price.
