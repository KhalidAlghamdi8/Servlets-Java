package com.examples.lesson07.bean;

public class NameBean {

  private String name;

  public String getName() {
    return name == null ? "world" : name;
  }

  public void setName(String name) {
    this.name = name;
  }
}
This is a simple JavaBean that has properties set by the helloJspXml.jspx page. If the name is null, then the default 'world' is returned in the getName method.
