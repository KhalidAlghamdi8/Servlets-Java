Expression Language functions are part of a Tag Library. As part of it, the taglib must be specified to be used using a JSP directive.
The JSTL includes a set of Expression Language functions that may be used whenever additional functionality that is not provided is needed by the core tags.
Functions in Expression Language are used with a prefix just like custom tags. The JSTL Expression Language functions Tag Library includes an assortment of functions that help manipulate data in JSP pages.
In this case, the function split(string, string) lets you split a given string based on the delimiter defined in the second.
Remember that string literals in Expression Language are defined by both " and '
In Expression Language you may obtain the size of a collection or an array using the function: fn:length(collection|array|string).
This function returns the size or length of the object.
Sometimes you may encounter string literals that contains > or < inside.
The function escapeXml(string) replces these characters by their XML equivalents: &gt; &lt;
