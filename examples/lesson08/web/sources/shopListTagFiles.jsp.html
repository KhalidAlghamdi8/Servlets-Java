<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<%@ taglib prefix="template" tagdir="/WEB-INF/tags" %>
<%@page contentType="text/html" pageEncoding="UTF-8"%>
<template:templateHeader title="Shopping list"/>

<table>
  <thead>
    <tr>
      <th>Item</th>
      <th>Quantity</th>
    </tr>
  </thead>
  <tbody>
    <c:forEach items="${sessionScope.shopList.items}" var="item" varStatus="status">
      <tr>
        <td>${item.name}</td>
        <td>${item.count}</td>
      </tr>
    </c:forEach>
  </tbody>
</table>

<template:templateFooter>
  Using JSP tag files!
</template:templateFooter>
When using custom tag files the taglib directive must be added. The difference with other taglib directives is that instead of using the URI attribute, the tagdir attribute is used.
This is a path where the tag files are. Tag files are special JSP files stored in the WEB-INF/tags folder (or any subfolder there) and inside JAR files under the META-INF/tags folder.
To use tag files, use the name of the tag file with the prefix just as with any other custom tag.
Like other tags, file tags may have attributes and body.
The header tag does not have a body and allows one attribute named 'title'
The footer tag allows a body but has no attributes.
