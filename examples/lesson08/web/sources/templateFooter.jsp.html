<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<%@page contentType="text/html" pageEncoding="UTF-8"%>
<!DOCTYPE html>
<c:url value="/index.jsp" var="home"/>
<a href="${home}"> Go home </a>
<h4>${param.footerNote}</h4>
</body>
</html>
This page is included at the end of the including page. This adds a link to "Go back" string and closes the body and HTML tag.
Although this is a footer page, it is an independent Java class (remember: JSPs are translated into Servlets) since it still requires JSP directives.
To use the Java Standard Tag Library Core tags, use the following JSP taglib directive: 
<%@page contentType="text/html" pageEncoding="UTF-8"%>
Parameters may be sent to the included pages using c:param if a c:import tag was used; or using jsp:param if a jsp:include tag was used.
To display parameters, use the ${param.paramName} expression.
The c:url tag sets the server-absolute path to a resourse inside a variable.
Here, the complete path is received (starting with a slash and including the context name) in the index.jsp page.
This path is used in the anchor tag.
