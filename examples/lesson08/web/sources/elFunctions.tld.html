The function element defines a custom Expression Language function inside a TLD. For a function, you need: the name of the function (to use in Expression Language), the Java class and the signature of the method.
Note that you need to use fully classified names.
If there methods with parameters, the correct signature for the method must be specified.
Note that only the type of the parameters is needed (no parameter names).
