<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<%@page contentType="text/html" pageEncoding="UTF-8"%>
<!DOCTYPE html>
<c:url value="/res/styles.css" var="stylesUrl"/>
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <link rel="stylesheet" href="${stylesUrl}" type="text/css"> 
    <title>${param.title}</title>
  </head>
  <h1>${param.title}</h1>
This JSP page contains the upper elements of an HTML page. The head and opening body tag are rendered in the page.
Noticee that dynamic elements may be used in included pages just like with other JSP pages. To use the Java Standard Tag Library Core tags, use the following JSP taglib directive: 
<%@page contentType="text/html" pageEncoding="UTF-8"%>
This page displays the title using a parameter provided from the including page. This is done by using the jsp:include tag with jsp:param tags or using the c:import tag with c:param tags.
The c:url tag sets the absolute URL for a context-relative URL. This is the preferred way of getting the path to a resource in your web application. Notice that the URL variable is displayed later in this page to include the CSS file. 
<link rel="stylesheet" href="${stylesUrl}" type="text/css">
