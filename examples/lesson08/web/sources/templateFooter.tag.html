<%@tag body-content="scriptless" %>
<!DOCTYPE html>
<c:url value="/index.jsp" var="home"/>
<a href="${home}"> Go home </a>
<h4><jsp:doBody/></h4>
</body>
</html>
The tag directive specifies properties for the tag. In this case, this tag specifies the body content that is used in the tag.
Custom tags does not allow JSP Java scriptlets as the body -- only Expression Language and JSP custom tags may be used.
The Jsp:doBody tag invokes the body of the tag where this tag is located.
